用户路径
同一用户可以是学习者，也可以是创作者。

2.1 学习者路径
1. 发现：
○ 浏览合集库 -> 查看合集信息（封面、简介、是否公开/免费）
2. 获取：
○ 免费合集 -> 收藏或添加到学习计划
○ 付费合集 -> 收藏（购买逻辑暂不实现）
3. 个性化：
○ 对卡片添加笔记（文本/手写）
○ 跟踪完成情况（打卡）
4. 坚持：
○ 根据学习计划完成每日任务
○ 系统记录完成量和用时
5. 分享：
○ 分享合集、章节或卡片链接
○ 链接附带来源信息、受访问权限控制

2.2 创作者路径
1. 创建合集：
○ 新建合集 -> 可选创建章节（最多一层子合集）
○ 添加卡片到合集（从别的合集添加），或者创建卡片 -> 支持多种官方模板卡片混合
2. 创建卡片：
○ 选择官方模板 -> 填写内容 -> 保存卡片
3. 管理卡片：
○ 同一张卡片只能属于一个合集或章节
○ 支持排序及编辑内容
○ 付费合集可部分可见
4. 发布/分享：
○ 设置合集可见性（私有/免费）
○ 分享合集或章节链接
